# taskworker-analyzetext
TaskBridge worker for chatting with texts via large language models (LLM)

## Installation

- Install Python 3.11
- Install Ollama

```sh
ollama pull llama3.2
python3 -m venv venv
venv\Scripts\activate
pip install ollama
```

## Running

```sh
venv\Scripts\activate
python analyzetext.py
```
